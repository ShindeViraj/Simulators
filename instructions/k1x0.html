PLC Instruction: MOV KnX0 D0

Instruction Config (Kn)
K4 = 16 Bits (Word)
Register D0 Value
Decimal:
0
Hexadecimal:
0000h
Logic:
The instruction groups input bits (X) into a value and moves it to register (D0).

K1 = 4 bits (X0-X3)
K2 = 8 bits (X0-X7)
K3 = 12 bits (X0-X11)
K4 = 16 bits (X0-X15)
